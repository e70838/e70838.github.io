# vi
## .bashrc:
`alias vi=vim`

## ~/.vimrc
```
set hls is noswf " nu
set listchars=tab:>-,trail:.
set list
highlight BadWhitespace ctermbg=red guibg=red
match BadWhitespace /\s\+$\|\t/
autocmd BufEnter * match BadWhitespace /\s\+$\|\t/
```

Le `set hls` permet d'highlighter tous les résultats de search.
Le `set ic` permet d'afficher les résultats de search de manière incrémentale quand on tappe.
Le `set noswf` évite la création d'un swap file.

Le `set list` traite les caractères listés dans listchars. `set nolist` pour ne plus faire les remplacements visuels.
Les lignes suivantes permettent d'afficher en rouge les espaces problèmatiques (espaces en fin de ligne et tabulations).

`set ic` rend les recherches non sensibles à la casse.
`set nu` affiche les numéros de ligne.

[sources](https://swordandsignals.com/2020/12/13/5-lines-in-vimrc.html)

# emacs
Pour faire l'équivalant:
```
;; Afficher les espaces en fin de ligne et les tabulations en rouge
(require 'whitespace)
(setq whitespace-style '(face trailing tabs tab-mark))
(set-face-attribute 'whitespace-trailing nil :background "red")
(set-face-attribute 'whitespace-tab nil :background "red")
(global-whitespace-mode 1)
```

# bash
Pour trouver les fichiers qui contiennent des espaces en fin de ligne:
`find . -name \*.java | xargs grep -Pn " +$"`

Les supprimer:
`sed -i 's/[[:space:]]*$//' fichier.java`

Pour trouver les fichiers qui contiennent des tabulations:
`grep -Pn "\t" fichier.java`
